## 🚀 Getting Started

**1. Clone the repository**

```bash
git clone https://github.com/mugged3000/focus-dev.git
cd mydocument
```

**2. Configure environment variables**

```bash
cp .env.example .env
# Edit .env with your settings for Contentful / ImageKit
```

**3. Install dependencies**

```bash
npm install

```


**4. Run development server:**

```bash
npm run dev
```
